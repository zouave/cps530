What new tag did you try? What was its effect?

  1. New tags that I used were: ol, ul, h1, h2, h3, hr.
  2. Effects for ol was that it listed with numbers unlike ul which is an unstructred list with circle shapes replacing numbers. h1, h2, h3 basically kept the fonts bigger and the size decreased as the number increased. Lastly hr create a horizontal line at the point where the tags were used.
